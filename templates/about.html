Analysis for Downtown Toronto

Using multiple open datasets, analysis for about 400,000 office and residential buildings was performed. Here, a subset of that analysis is provided which focuses on the dense urban region of Downtown Toronto. This analysis hopes to shed light on the scale of infrastructure required for the future of such grids. This is helpful for urban planners, investors, climate change enthusiasts, and those fascinated by cities - including yours truly.

How to Use

To use the tool, provide a unique building name and Gross Floor Area for each property you want to map. Enter the latitude and longitude obtained from Google Maps. Also, provide the building type. If this is not provided, the building is considered to be an Office. Press "Add Row" to enter more buildings.

Technical Info

The groups and sub groups are generated using DBSCAN from Python's scikit-learn package. The tool uses Random Forests and AdaBoost Machine Learning models to predict the energy consumption for a given hour using the weather data for the last 10 hours. The model scores up to 40%, which is quite significant considering that the only building information provided is area and type. Moreover, by limiting the analysis to short time series (less than a day), the model foregoes the traditional time series analysis.
