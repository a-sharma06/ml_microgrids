Machine Learning Driven Microgrids - Demo

Predicting Energy Demand for Urban Centers Using Machine Learning

A microgrid is an electrical network of buildings located close to each other. This grid allows the buildings to meet each other's energy demand, before seeking energy from the main grid.

About the Tool

The goal of this tool is to allow the user to enter some building properties and generate a microgrid for the specified buildings. Then, the energy consumption is estimated for these buildings during extreme weather conditions (like snow, storms etc). As a result, a network of microgrid optimized for extreme weather is generated. The default location values correspond to a few NYU nuildings close to Washington Square Park in Manhattan, NYC.
